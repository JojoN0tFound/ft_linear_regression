# ft_linear_regression

## usage:
```
./train.py   [-h] [-v] [-ngd]
./predict.py [-h] [-r] [-cmp]
```

## dependencie:
```
pip3 -> (sudo) apt install python3-pip
venv -> (sudo) apt-get install python3-venv
```
**check for update with apt-get update**

## before running:
```
python3 -m venv env
source env/bin/activate
pip install -r requirements.txt
```
